mixme.tf

mixme.tf is a webpage that I made for mixing a Team Fortress 2 6s team. It simply takes six names and with the click of a button it randomly assigns the roles of the 6s gamemode to the names that were input. In addition to that it has some links to Websites that are useful for mixing, a copy box that includes which player got assigned to what class and a 'GET PROOF'-Button that screenshots the table.
